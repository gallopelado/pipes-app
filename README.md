# PipesApp -Trabajar con Pipes de Angular

Este proyecto se generó con [Angular CLI](https://github.com/angular/angular-cli) versión `19.2.19`.

## Development server

Para arrancar el servidor de desarrollo, ejecutar la sentencia:

```bash
ng serve
```

Estará listo en `http://localhost:4200/`.

### Algunos paquetes instalados

- [Tailwindcss `V3.4.17`](https://v3.tailwindcss.com/docs/guides/angular)
- [DaisyUI `V4.12.24`](https://v4.daisyui.com/)
